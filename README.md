# music-desktop

![](https://img.shields.io/badge/react-javascript-brightgreen.svg)
![](https://img.shields.io/badge/electron-web-brightgreen.svg)

electron cross-platform desktop music application
使用 JavaScript、HTML 和 CSS 构建跨平台的桌面应用

## 使用 create-react-app

```bash
npm install -g create-react-app

npx create-react-app foldName

npm run eject
```

## 配置 less

```bash
npm install less-loader less --save-dev
```

## 使用 react-id-swiper

```bash
npm install --save react-id-swiper

// 引入 swiper 样式
import 'assets/css/swiper.css'
```

## 使用 Electron

```bash
npm install electron --save-dev

npm run elec-start

// 配置 main.js
vi electron/main.js
```

## 使用 react-router

```bash
npm install --save react-router-dom
```

## 启动项目

```bash
// 启动 web
npm start

// 启动 electron
npm run elec-start
```
